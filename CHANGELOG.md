## [1.0.0] - 2026-03-07

### Added
- Initial release of Aegis framework
- MCP Server for tool orchestration
- CLI interface with argparse
- Nmap port scanning integration
- AI agent with risk analysis
- Comprehensive documentation
- INSTRUCTIONS.md for tool development
- CONTRIBUTING.md for developers
- CODE_OF_CONDUCT.md for community
- Rich terminal formatting
- Full type hints throughout
- Error handling and validation

### Features
- Port scanning with service detection
- Risk assessment (CRITICAL, HIGH, MEDIUM, LOW)
- Security recommendations
- Extensible tool architecture
- Beautiful CLI output with tables
- Python API for integration
- MCP server for AI agent integration
